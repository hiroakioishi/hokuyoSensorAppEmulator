# 北陽電機測域センサーアプリエミュレータ
Hokuyo LiDAR Sensor App Emulator

## Processing version
4.5.2
## Required Libraries（必要なライブラリ）
### oscP5
An Open Sound Control(OSC)implementation.

## How to install libraries（ライブラリのインストール方法）
1. Processing上部メニュー<br/>スケッチ > ライブラリをインポート > **Manage Libraries**<br/>を押し、「Contribution Manager」を開く。
2. 「oscP5」を検索し、**Install**ボタンを押す。
